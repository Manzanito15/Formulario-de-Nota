Este es mi programa de C# Windows Forms .Net este es un programa que tiene la funcion de calcular las 4 notas de el estudiante y si el estudiante tiene una nota menor a 70 teve ir a completivo se va habilitar el texbox de completivo y que pueda ingresar la nota que saco el completivo el caso de q sea menor a 70 tiene que ir a extradinario si saca menos de 70 esta reprobado el caso de que sace mas o igual a 70 apravodo el extradonario este tambien cuenta con un boton de limpiar que este limpia los campos que esten en el formulario para dejarlo para que se pueda volver a utilizar.

Tecnologia utilizadas: Estuve utilizando el IDE Microsft Visual Studio 2022 y el lengua de programación C#.

Herramientas utilizadas: Aplicación de Windows Forms .Net.
