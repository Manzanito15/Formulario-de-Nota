# Nombre del Proyecto:  Sistema de Cálculo de Notas
### Sistema de Cálculo de Notas

## Descripción

#### Este proyecto consiste en una aplicación de escritorio desarrollada en C# que permite capturar las cuatro notas de un estudiante, calcular el promedio y determinar si el estudiante aprueba, va a completivo, extraordinario o reprueba, aplicando las reglas establecidas para cada caso.

## Tecnologías Utilizadas

Lenguaje: C#

Herramientas / Framework: Windows Forms (.NET Framework)

Base de datos: No aplica

Entorno o IDE: Visual Studio

## Uso o Ejecución

Abrir el proyecto en Visual Studio.

Ejecutar la aplicación presionando la tecla F5.

Ingresar las cuatro notas del estudiante.

Presionar el botón Calcular.

Si corresponde, ingresar la nota del completivo y/o extraordinario.

El sistema mostrará automáticamente el resultado final.
